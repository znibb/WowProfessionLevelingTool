Working principles:
  1. Only crafts with guaranteed skillup are considered
  2. The calculation is iterative, meaning it will always calculate the cheapest way to go from skill 'x' to 'x+1'
  3. Only the cost of purchasing reagents are considered when determining the cheapest skillup, i.e. the price to learn the skill from the trainer or purchase the recipe isn't included (uses undiscounted vendor price for items available from vendors)
  4. The algorithm terminates if there aren't any available crafts for the current skill level. This means if the listed guide doesn't reach to the target level you entered you probably need to include another source for there to be a solution
  5. If you need e.g. Bolt of Runecloth for a craft it'll check the price for both Runecloth and Bolt of Runecloth and add the option that's cheaper to the shopping list
